# Vue 3 Template

這是一個 Vue 3 的模板專案，用於快速起始新的專案用

## Service 的使用

Vue 中使用 Service 與 Angular 最大差異是其無法直接透過 class 宣告 Service，必須使用 const + 匿名函數方式宣告。
